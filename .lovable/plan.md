
# ระบบค้นหารายการที่เคยจด

## ภาพรวม
เพิ่มระบบค้นหาอัจฉริยะที่ช่วยให้ผู้ใช้ค้นหารายการที่เคยบันทึกไว้ได้อย่างรวดเร็ว โดยสามารถค้นหาจากคำอธิบาย หมวดหมู่ หรือจำนวนเงิน และนำรายการเดิมมาใช้เป็นแม่แบบสำหรับการเพิ่มรายการใหม่ได้

## ฟีเจอร์หลัก

1. **ปุ่มค้นหารายการเดิม** - เพิ่มในหน้าเพิ่มรายการ (AddTransaction)
2. **Dialog ค้นหาแบบ Command Palette** - ใช้ cmdk component ที่มีอยู่แล้ว
3. **การค้นหาอัจฉริยะ** - ค้นหาจากคำอธิบาย, หมวดหมู่, จำนวนเงิน
4. **การนำไปใช้** - เมื่อเลือกรายการเดิมจะกรอกข้อมูลในฟอร์มอัตโนมัติ
5. **แสดงรายการยอดนิยม** - แสดงรายการที่ใช้บ่อยเป็นอันดับแรก

## ขั้นตอนการพัฒนา

### ขั้นตอนที่ 1: สร้าง Component ค้นหารายการ
สร้าง `TransactionSearch.tsx` ที่ใช้ Command Dialog เพื่อค้นหารายการเดิม

**ความสามารถ:**
- ค้นหาแบบ real-time ขณะพิมพ์
- แสดงรายการที่ใช้บ่อย (จัดกลุ่มตาม description)
- แสดงข้อมูลสำคัญ: คำอธิบาย, หมวดหมู่, จำนวนเงิน, ประเภท (รายรับ/รายจ่าย)
- รองรับทั้งภาษาไทยและอังกฤษ

### ขั้นตอนที่ 2: สร้าง Hook สำหรับค้นหา
สร้าง `useTransactionSearch.ts` เพื่อจัดการการค้นหาและจัดกลุ่มรายการ

**ความสามารถ:**
- ดึงรายการทั้งหมดจาก Supabase
- จัดกลุ่มรายการที่ซ้ำกัน
- นับจำนวนครั้งที่ใช้แต่ละรายการ
- เรียงลำดับตามความนิยม

### ขั้นตอนที่ 3: เพิ่มปุ่มค้นหาในหน้าเพิ่มรายการ
แก้ไข `AddTransaction.tsx` เพื่อเพิ่มปุ่มค้นหารายการเดิม

**UI เพิ่มเติม:**
- ปุ่ม "ค้นหารายการเดิม" ด้านบนฟอร์ม
- เมื่อเลือกรายการ กรอกข้อมูลอัตโนมัติ: ประเภท, จำนวนเงิน, หมวดหมู่, คำอธิบาย

## รายละเอียดทางเทคนิค

### 1. TransactionSearch Component

```text
+----------------------------------------------+
|  🔍 ค้นหารายการที่เคยบันทึก...               |
+----------------------------------------------+
| รายการที่ใช้บ่อย                              |
| +------------------------------------------+ |
| | ☕ กาแฟ                  รายจ่าย  ฿45   | |
| | (ใช้ 15 ครั้ง)          อาหาร           | |
| +------------------------------------------+ |
| | 🚕 ค่าแท็กซี่            รายจ่าย  ฿150  | |
| | (ใช้ 8 ครั้ง)           ค่าเดินทาง       | |
| +------------------------------------------+ |
| | 💰 เงินเดือน             รายรับ  ฿35,000| |
| | (ใช้ 6 ครั้ง)           เงินเดือน        | |
| +------------------------------------------+ |
+----------------------------------------------+
```

### 2. ข้อมูลที่เก็บจากการค้นหา

```typescript
interface SearchableTransaction {
  description: string;
  type: 'income' | 'expense';
  amount: number;
  category: string;
  usageCount: number; // จำนวนครั้งที่ใช้
}
```

### 3. การจัดกลุ่มและนับความถี่

- จัดกลุ่มตาม `description` (คำอธิบาย)
- นับจำนวนครั้งที่มีการบันทึกรายการซ้ำ
- เรียงลำดับตามความถี่ (มากไปน้อย)
- แสดงจำนวนเงินล่าสุดที่ใช้

### 4. ไฟล์ที่ต้องสร้าง/แก้ไข

| ไฟล์ | การดำเนินการ |
|------|-------------|
| `src/components/TransactionSearch.tsx` | สร้างใหม่ |
| `src/hooks/useTransactionSearch.ts` | สร้างใหม่ |
| `src/components/screens/AddTransaction.tsx` | แก้ไข |

## การรองรับภาษา

เพิ่มข้อความแปลใน component:
- **ไทย**: "ค้นหารายการเดิม", "รายการที่ใช้บ่อย", "ใช้ X ครั้ง", "ไม่พบรายการ"
- **อังกฤษ**: "Search previous entries", "Frequently used", "Used X times", "No entries found"

## ประโยชน์

1. **ประหยัดเวลา** - ไม่ต้องพิมพ์ข้อมูลซ้ำทุกครั้ง
2. **ลดข้อผิดพลาด** - ใช้ข้อมูลเดิมที่เคยบันทึกไว้
3. **สะดวกใช้งาน** - ค้นหาได้รวดเร็วด้วย Command Palette
4. **เข้าถึงง่าย** - แสดงรายการยอดนิยมทันทีที่เปิด
